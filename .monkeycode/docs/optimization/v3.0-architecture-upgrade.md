## 1. 背景与研究

### 1.1 竞品分析总结

| 工具 | 定位 | 架构特点 | ApiRed 差距 |
|------|------|----------|-------------|
| **Noir (OWASP)** | SAST-to-DAST 桥接 | Crystal 语言，LLM 驱动端点发现 | 缺乏 LLM 驱动的智能发现 |
| **Akto** | API 安全测试平台 | 1000+ 测试用例，CI/CD 集成 | 测试用例库不足，无 CI/CD 支持 |
| **Metlo** | 流量分析 | Go 实现，OpenAPI 自动生成 | 需部署 agent，纯黑盒能力弱 |
| **Cherrybomb** | OpenAPI 验证 | Rust 实现，Brewery DSL | 仅验证已有规范，缺乏发现能力 |

### 1.2 当前项目问题

#### P0 - 阻塞问题（必须立即修复）

| 问题 | 位置 | 影响 |
|------|------|------|
| `_stage2-4` 是 stub | `core/scanner.py:193,208,223` | **核心扫描功能不可用** |
| Agent AI 抽象为空 | `core/agents/base.py` | AI 增强功能形同虚设 |
| SSL 验证禁用 | `core/utils/http_client.py:73-75` | **安全风险：中间人攻击** |

#### P1 - 严重问题（影响功能）

| 问题 | 位置 | 影响 |
|------|------|------|
| Scanner/Pipeline 冗余 | `scanner.py` vs `pipeline.py` | 代码维护困难 |
| Config 线程不安全 | `core/utils/config.py` | 多线程环境竞态条件 |
| 测试覆盖不全 | `testers/` | 仅 SQLi/XSS/SSRF，缺 OWASP Top10 |

#### P2 - 优化问题（体验提升）

| 问题 | 位置 | 改进方向 |
|------|------|----------|
| 响应聚类 O(n^2) | `response_cluster.py` | 优化算法复杂度 |
| JS 解析仅正则 | `js_collector.py` | 引入 AST 解析（esprima） |
| 报告格式单一 | `exporters/` | 增加 Markdown/CSV 支持 |

```python
        # 单一入口，统一流程
```

#### 3.3.2 Agentic AI Layer

```python
# core/agents/base.py (重构)
class BaseAgent(ABC):
    """Agent 基类，完整实现"""
    
    def __init__(self, llm_client: LLMClient):
        self.llm = llm_client
        self.memory = AgentMemory()
        
    async def think(self, prompt: str, context: Dict) -> str:
        """思考：分析当前状态，生成下一步计划"""
        pass
    
    async def act(self, plan: str, context: Dict) -> Action:
        """行动：执行计划，返回结果"""
        pass
    
    async def reflect(self, result: Any, context: Dict) -> None:
        """反思：学习经验，更新记忆"""
        pass

class ScannerAgent(BaseAgent):
    """扫描 Agent：智能路径发现"""
    
    async def analyze_js(self, js_content: str) -> List[APIEndpoint]:
        """从 JS 中发现 API 端点"""
        pass
    
    async def fuzz_paths(self, base_url: str, hints: List[str]) -> List[APIEndpoint]:
        """智能 Fuzz：基于 LLM 生成候选路径"""
        pass

class AnalyzerAgent(BaseAgent):
    """分析 Agent：漏洞研判"""

        """评估端点风险等级"""
        pass
    
    async def suggest_tests(self, endpoint: APIEndpoint) -> List[TestCase]:
        """推荐测试用例"""
        pass

class TesterAgent(BaseAgent):
    """测试 Agent：漏洞验证"""
    
    async def generate_payloads(self, vuln_type: str) -> List[Payload]:

        """验证漏洞是否存在"""
        pass
```
